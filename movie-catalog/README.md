Повноцінний веб-застосунок для керування каталогом фільмів. 
Проєкт реалізований на стеку NestJS (Backend) та React (Frontend).

Технологічний стек
Frontend:
React 18 (Vite)

TypeScript

Axios (для запитів до API)

React Router Dom (навігація)

CSS3 (адаптивна сітка та стилізація)

Backend:
NestJS

TypeScript

Class-validator / Class-transformer (валідація даних)

In-memory storage (робота з даними без SQL на даному етапі)

 Функціонал
Головна сторінка: Відображення списку фільмів у вигляді карток з постерами, рейтингом та короткою інформацією.

Детальна сторінка: Перегляд повного опису фільму, жанру та року випуску.

Керування даними (CRUD):

Створення нового фільму через форму (валідація полів).

Видалення фільмів з каталогу.

Можливість додавання посилань на постери.

Валідація: Обробка помилок при введенні (наприклад, перевірка числових значень рейтингу через крапку).

 Інструкція з встановлення та запуску
1. Клонування проєкту
Bash
git clone https://github.com/matviybichevyi-gif/movie-catalog
cd movie-catalog
2. Запуск Backend
Відкрийте термінал у папці backend:

Bash
cd backend
npm install
npx ts-node src/main.ts
Сервер запуститься на http://localhost:3001

3. Запуск Frontend
Відкрийте новий термінал у папці movie-catalog:

Bash
npm install
npm run dev
Застосунок буде доступний за адресою http://localhost:5173

 Структура проєкту
/backend/src/movies — логіка обробки фільмів (сервіси, контролери, DTO).

/movie-catalog/src/pages — основні сторінки (HomePage, MovieDetail, AddMovie).

/movie-catalog/src/components — багаторазові компоненти інтерфейсу.
